## 6Ô∏è‚É£ Jalankan Server

1. Buka terminal di root project
2. Jalankan built-in PHP server:

```bash
php -S localhost:8888 -t public
```

3. Buka browser:

```
http://localhost:8888/
```

* Ini akan menampilkan halaman utama daftar produk
* Route lain:

  * `/auth/signin` ‚Üí login
  * `/product/{id}` ‚Üí detail produk
  * `/order/{id}` ‚Üí ringkasan order
  * `/dashboard` ‚Üí dashboard (login dulu)
  ...

## 8Ô∏è‚É£ Alur Pemesanan & Pembayaran

1. **Daftar Produk** ‚Üí `/`
2. **Detail Produk** ‚Üí `/product/{id}` ‚Üí pilih jumlah ‚Üí klik **Beli Sekarang**
3. **Ringkasan Order** ‚Üí `/order/{id}` ‚Üí pilih metode pembayaran ‚Üí klik **Checkout Sekarang**
4. **Payment Mock**:

   * Status otomatis: `pending` ‚Üí bisa disimulasikan `success` / `failed`
   * Data payment tersimpan di tabel `payments`
5. **Order**:

   * Tersimpan di tabel `orders`
   * Status order: `pending` ‚Üí update `success` / `failed` sesuai payment
